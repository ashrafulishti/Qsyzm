## QSYZM — Zero Knowledge Cipher System
A dual-layer, zero-knowledge encryption web service. XOR cascade through a user-selected 3-digit route, wrapped in AES-256-GCM with PBKDF2 key derivation. 
- The server never stores or logs your route or password — ever.
